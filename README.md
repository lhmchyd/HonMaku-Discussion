**Honmaku** is a simple, privacy-friendly anime tracker built with [Svelte](https://svelte.dev) and [shadcn-svelte](https://ui.shadcn.com).  
It’s currently under active development and aims to provide a fast, elegant, and login-free way to manage your anime list — all stored locally in your browser.

This repository is for **discussions, feedback, and community interaction** about Honmaku.  
The source code is private and not open for public development at this time.

## Overview

Honmaku is built to give anime fans a clean and personal way to track their shows — without accounts, tracking, or ads.  
All progress is stored locally in your browser, giving you full control of your data.

### Key Features

- Local anime tracking using **localStorage**  
- **Optional GitHub login** for commenting via [Giscus](https://giscus.app)  
- Comments are **moderated by GitHub**, not manually by this project  
- Default **dark mode** interface  
- **Export/Import** support with [MyAnimeList](https://myanimelist.net)  
- **No ads, no analytics, no tracking**  
- Free to use, community-driven discussions  

> Sync between devices is not yet supported. This feature is planned for a future release.

## Comment & Discussion Guidelines

To keep the community open, friendly, and relevant, please follow these guidelines:

1. Be respectful — avoid harassment or personal attacks.  
2. Stay constructive — feedback is welcome, negativity is not.  
3. You may talk about **anime**, **manga**, **related games**, or other **relevant topics**.  
   - Occasional off-topic (OOT) discussions are fine as long as they stay within the general theme.  
4. Avoid spam, advertising, or unrelated self-promotion.  
5. Discussions are public and follow GitHub’s moderation policies.  

Comments are managed through Giscus and are subject to GitHub’s content moderation.
